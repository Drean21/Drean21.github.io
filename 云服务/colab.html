需要注意的事项：

  1. colab默认可直接运行python脚本语言并内置安装了特定版本的python

  2. 使用前请先切换笔记本的类型选择T4的GPU运行环境

  3. 作为可交互环境，在命令前面加上英文!既可以使用Linux系统命令

  4. 但是要注意目录的切换：

    1. 方法一，用Linux的cd命令，在colab中是可以运行Linux命令的，只需要在命令前加一个感叹号“!

      !cd /content/sample_data
!pwd
//不过这个方法有时会失效，此时换第二种方法

    2. 方法二，直接用os.chdir()指令

      import os
path = "/content/sample_data"
os.chdir(path)
print(os.getcwd())
//亲测有效
